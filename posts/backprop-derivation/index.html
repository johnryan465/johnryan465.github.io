Introduction

First we must define how a feed-forward network functions.

The inputs to the model can be viewed as the 0th layers output. ϕ0=X\mathbf{\phi}^0 = \mathbf{X}ϕ0=X

Note: the indexing of the weight matrix may seem backwards but doing it this way prevents us having to take the transpose of the weight matrix when converting to matrix form.

ρjL=∑kWjkLϕkL−1+BjLρL=WLϕL−1+BLϕjL=σ(∑kWjkLϕkL−1+BjL)ϕL=σv(ρL)\begin{aligned} \mathbf{\rho}^{L}_j &= \sum_k W^L_{jk}\phi^{L-1}_k + B^L_j \\ \mathbf{\rho}^L &= W^L\phi^{L-1} + B^L \\ \mathbf{\phi}^{L}_j &= \sigma \left(\sum_k W^L_{jk}\phi^{L-1}_k + B^L_j\right)\\ \mathbf{\phi}^L &= \sigma_v \left(\rho^L\right)\\ \end{aligned}ρjL​ρLϕjL​ϕL​=k∑​WjkL​ϕkL−1​+BjL​=WLϕL−1+BL=σ(k∑​WjkL​ϕkL−1​+BjL​)=σv​(ρL)​

Given all of the required parameters are given and are properly defined this fully defines a feed-forward network. These definition are not where the magic of a neural net happens (I intend on detailing how the formulation allows it to be a good function approximator however at some point).

The magic happens by combining an optimisation algorithm (gradient descent is the standard) with backpropagation.

Derivation

Backprop sounds very mysterious but all it is, is an algorithm to calculate partial derivatives of the cost function with respect to each of the learnable parameters (the weights and biases). ie ∂C∂WjkL,∂C∂BjL\frac{\partial C}{\partial W_{jk}^L} , \frac{\partial C}{\partial B_j^L}∂WjkL​∂C​,∂BjL​∂C​

The most common cost function is the squared error (below), but many others exist. C=12n∑i(ϕl(xi)−yi)2C = \frac{1}{2n}\sum_i\left(\phi^l(x_i)-y_i\right)^2C=2n1​∑i​(ϕl(xi​)−yi​)2

∂C∂WjkL=∂C∂ϕjL∂ϕjLWjkL=∑a,b,…,f∂C∂ϕal∂ϕal∂ϕbl−1…∂ϕfL+1∂ϕjL∂ϕjLWjkL\begin{aligned}\frac{\partial C}{\partial W_{jk}^L} &= \frac{\partial C}{\partial \phi^L_j} \frac{\partial \phi^L_j}{W_{jk}^{L}} \\ &= \sum_{a,b,\ldots,f} \frac{\partial C}{\partial \phi^l_a}\frac{\partial \phi^{l}_a}{\partial \phi^{l-1}_b}\ldots\frac{\partial \phi^{L+1}_f}{\partial \phi^{L}_j} \frac{\partial \phi^L_j}{W_{jk}^{L}} \\ \end{aligned}∂WjkL​∂C​​=∂ϕjL​∂C​WjkL​∂ϕjL​​=a,b,…,f∑​∂ϕal​∂C​∂ϕbl−1​∂ϕal​​…∂ϕjL​∂ϕfL+1​​WjkL​∂ϕjL​​​

We now have a formulation for the result we wish to calculate, however computing this directly for all of the nodes in the network would be quite expensive.

From this formulation it makes sense that we will try to compute the derivatives from the last layer back.

We will attempt to derive the derivatives of the current layer in terms of the previous

We will define ωjL=∂C∂ϕjL\omega^L_j = \frac{\partial C}{\partial \phi^L_j} ωjL​=∂ϕjL​∂C​, this is the error of the output of the jth node in the Lth layer.

∂C∂ϕjL=∑a∂C∂ρaL+1∂ρaL+1∂ϕjLωjL=∑a∂C∂ρaL+1WajL+1=∑a∂C∂ϕaL+1∂ϕaL+1∂ρaL+1WajL+1=∑a∂C∂ϕaL+1σ′(ρaL+1)WajL+1=∑aωaL+1σ′(ρaL+1)WajL+1=∑a(ωL+1∘σv′(ρL+1))aWajL+1=∑a(W∗,jL+1)a(ωL+1∘σv′(ρL+1))a=(W∗,jL+1)T(ωL+1∘σv′(ρL+1))\begin{aligned} \frac{\partial C}{\partial \phi^L_j} &= \sum_a \frac{\partial C}{\partial \rho^{L+1}_a}\frac{\partial \rho^{L+1}_a}{\partial \phi^L_j} \\ \omega_j^L &= \sum_a \frac{\partial C}{\partial \rho^{L+1}_a} W_{aj}^{L+1} \\ &= \sum_a \frac{\partial C}{\partial \phi^{L+1}_a} \frac{\partial \phi_{a}^{L+1}}{\partial \rho^{L+1}_a} W_{aj}^{L+1} \\ &= \sum_a \frac{\partial C}{\partial \phi^{L+1}_a} \sigma'\left( \rho^{L+1}_a \right) W_{aj}^{L+1} \\ &= \sum_a \omega^{L+1}_a \sigma'\left( \rho^{L+1}_a \right) W_{aj}^{L+1} \\ &= \sum_a \left (\omega^{L+1} \circ \sigma_v'\left( \rho^{L+1} \right) \right)_{a}W_{aj}^{L+1} \\ &= \sum_a \left(W_{*,j}^{L+1}\right)_a \left (\omega^{L+1} \circ \sigma_v'\left( \rho^{L+1} \right) \right)_{a} \\ &= \left(W_{*,j}^{L+1}\right)^T \left (\omega^{L+1} \circ \sigma_v'\left( \rho^{L+1} \right) \right) \\ \end{aligned}∂ϕjL​∂C​ωjL​​=a∑​∂ρaL+1​∂C​∂ϕjL​∂ρaL+1​​=a∑​∂ρaL+1​∂C​WajL+1​=a∑​∂ϕaL+1​∂C​∂ρaL+1​∂ϕaL+1​​WajL+1​=a∑​∂ϕaL+1​∂C​σ′(ρaL+1​)WajL+1​=a∑​ωaL+1​σ′(ρaL+1​)WajL+1​=a∑​(ωL+1∘σv′​(ρL+1))a​WajL+1​=a∑​(W∗,jL+1​)a​(ωL+1∘σv′​(ρL+1))a​=(W∗,jL+1​)T(ωL+1∘σv′​(ρL+1))​

This shows that we can compute the error of the previous layer from the current one.

∂C∂ϕL=(WL+1)T(ωL+1∘σv′(ρL+1))(A1)\begin{aligned} \frac{\partial C}{\partial \phi^L} &= \left(W^{L+1}\right)^T \left (\omega^{L+1} \circ \sigma_v'\left( \rho^{L+1} \right) \right) \\ \end{aligned} \tag{A1}∂ϕL∂C​​=(WL+1)T(ωL+1∘σv′​(ρL+1))​(A1)
∂C∂BjL=∂C∂ϕjL∂ϕjL∂BjL=ωjLσ′(ρjL)(1)=ωjLσ′(ρjL)\begin{aligned}\frac{\partial C}{\partial B^L_j} &= \frac{\partial C}{\partial \phi_j^L}\frac{\partial \phi_j^L}{\partial B_j^L} \\ &= \omega_j^L\sigma'\left( \rho_j^L \right)(1)\\ &= \omega_j^L\sigma'\left( \rho_j^L \right) \\ \end{aligned}∂BjL​∂C​​=∂ϕjL​∂C​∂BjL​∂ϕjL​​=ωjL​σ′(ρjL​)(1)=ωjL​σ′(ρjL​)​
∂C∂BL=ωL∘σv′(ρL)(A2)\begin{aligned}\frac{\partial C}{\partial B^L} &= \omega^L \circ \sigma_v'\left(\rho^L\right) \tag{A2}\\ \end{aligned}∂BL∂C​​=ωL∘σv′​(ρL)​(A2)
∂C∂WjkL=∂C∂ϕjL∂ϕjL∂WjkL=ωjLσ′(ρjL)(ϕkL−1)\begin{aligned}\frac{\partial C}{\partial W^L_{jk}} &= \frac{\partial C}{\partial \phi_j^L}\frac{\partial \phi_j^L}{\partial W_{jk}^L} \\ &= \omega_j^L\sigma'\left( \rho_j^L \right)\left(\phi_k^{L-1}\right)\\ \end{aligned}∂WjkL​∂C​​=∂ϕjL​∂C​∂WjkL​∂ϕjL​​=ωjL​σ′(ρjL​)(ϕkL−1​)​
∂C∂WL=(ωL∘σv′(ρL))(ϕL−1)T(A3)\begin{aligned}\frac{\partial C}{\partial W^L} &= \left (\omega^L \circ \sigma_v'\left( \rho^L \right) \right)\left(\phi^{L-1}\right)^T \tag{A3}\\ \end{aligned}∂WL∂C​​=(ωL∘σv′​(ρL))(ϕL−1)T​(A3)

This allows us to compute the derivative with respect to each parameter we can control, if we compute the error terms first.

So in what order should we compute these values? We can see that we first should compute ρL,ϕL,σv′(ρL)\rho^L,\phi^L,\sigma_v'(\rho^L)ρL,ϕL,σv′​(ρL). This can be done in a single forward pass. Then we can compute all the error terms in a single backward pass.

This is all backpropagation is for a straightforward feed forward network.

Speed up

We can see that in our formulation that when ever we use ωL\omega^LωL, it is in the context of ωL∘σv′(ρL)\omega^L \circ \sigma_v' \left(\rho^L\right)ωL∘σv′​(ρL). We compute this quantity in all of the steps of the algorithm. We could define λL=ωL∘σv′(ρL)\lambda^L = \omega^L \circ \sigma_v' \left(\rho^L\right)λL=ωL∘σv′​(ρL)

ωL=(WL+1)TλL+1ωL∘σv′(ρL)=((WL+1)TλL+1)∘σv′(ρL)λL=((WL+1)TλL+1)∘σv′(ρL)(B1)\begin{aligned} \omega^L &= \left( W^{L+1}\right)^T \lambda^{L+1} \tag{B1}\\ \omega^L \circ \sigma_v' \left(\rho^L\right) &= \left(\left(W^{L+1}\right)^T \lambda^{L+1} \right)\circ \sigma_v' \left(\rho^L\right) \\ \lambda^L &= \left(\left(W^{L+1}\right)^T \lambda^{L+1} \right)\circ \sigma_v' \left(\rho^L\right) \\ \end{aligned}ωLωL∘σv′​(ρL)λL​=(WL+1)TλL+1=((WL+1)TλL+1)∘σv′​(ρL)=((WL+1)TλL+1)∘σv′​(ρL)​(B1)
∂C∂BL=ωL∘σv′(ρL)∂C∂BL=λL(B2)\begin{aligned}\frac{\partial C}{\partial B^L} &= \omega^L \circ \sigma_v'\left(\rho^L\right) \tag{B2}\\ \frac{\partial C}{\partial B^L} &= \lambda^L\\ \end{aligned}∂BL∂C​∂BL∂C​​=ωL∘σv′​(ρL)=λL​(B2)
∂C∂WL=(ωL∘σv′(ρL))(ϕL−1)T∂C∂WL=λL(ϕL−1)T(B3)\begin{aligned}\frac{\partial C}{\partial W^L} &= \left (\omega^L \circ \sigma_v'\left( \rho^L \right) \right)\left(\phi^{L-1}\right)^T \tag{B3}\\ \frac{\partial C}{\partial W^L} &= \lambda^L\left(\phi^{L-1}\right)^T \\ \end{aligned}∂WL∂C​∂WL∂C​​=(ωL∘σv′​(ρL))(ϕL−1)T=λL(ϕL−1)T​(B3)

NOTE

This version of the equations are probably the ones you have seen before. If we had defined ωjL=∂C∂ρjL\omega^L_j = \frac{\partial C}{\partial \rho^L_j}ωjL​=∂ρjL​∂C​ instead, our first set of derived equations would look like our modified version. This was done intentionally to show that even without that observation it can be derived.

While this algorithm is extremely powerful, it's nothing that a student with some linear algebra and multivariate calculus knowledge couldn't derive.
